# **App Name**: AgriAssist SMS

## Core Features:

- Weather Data Integration: Automatically fetches real-time weather data from reliable APIs. Connects to available APIs to act as tool to determine hyper-local forecasts.
- Crop Advisory Alerts: Sends SMS notifications with crop-specific advisory tips based on the weather conditions, sourced from government sources.
- Pest Outbreak Warnings: Delivers timely SMS alerts about potential pest outbreaks in the user's region.
- SMS Notification System: Manages and sends all SMS notifications to registered farmers.
- User Registration: Allows farmers to register via SMS, providing essential details such as location, crops grown, and preferred language.
- Preference Management: Enables users to update their crop preferences, location, and language via SMS commands.
- Localized Content Delivery: Delivers advisory tips and weather alerts in the farmer's local language for better comprehension.

## Style Guidelines:

- Primary color: Earthy green (#8FBC8F) to evoke nature and agriculture.
- Background color: Very light, desaturated beige (#F5F5DC).
- Accent color: Warm orange (#E67700) to highlight important information and calls to action. 
- Body and headline font: 'PT Sans', a humanist sans-serif font, for a modern yet warm and readable feel.
- Simple, easily recognizable icons to represent weather conditions and crop types.
- Clean and straightforward SMS layout for ease of understanding on basic mobile phones.
